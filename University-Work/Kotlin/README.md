# SpaceShooter

**SpaceShooter** is a 2D arcade-style game developed for Android as part of the **Game Development for Android** course. Players control a spaceship to navigate through space and shoot incoming enemies, aiming to achieve the highest score possible.

## Features

- Smooth player movement with responsive touch controls  
- Shooting mechanics to eliminate enemies  
- Collision detection between player, lasers, and enemies  
- Scoring system and level progression  
- Clean and modular code structure implemented in Kotlin

## Technologies

- **Language:** Kotlin  
- **IDE:** Android Studio  
- **Platform:** Android  

## Project Focus

The project emphasizes core game development concepts including game mechanics, modular design, and mobile-friendly controls. 
It demonstrates the implementation of a simple yet functional 2D arcade game suitable for Android devices.

## How to Run

1. Clone the repository  
2. Open the project in Android Studio  
3. Build and run on an Android device or emulator
